# Publishing Checklist (Safety)

- Verify no API keys/tokens/passwords in any file.
- Verify no private/internal URL is present.
- Verify no real customer data in screenshots/video.
- Verify sample CSV/API data is anonymized.
- Verify hardcoded IDs are replaced by config constants.
- Verify comments do not expose confidential supplier terms (if required).
- Verify README explains that this is a feature case, not full store source.
